# JavaSprits

# 2016级项目实训成果展示 

## 《教学辅助工具》 - 大数据

### 项目简介

一款旨在提高本科教学质量的教学辅助系统，基于软件学院的教学模式，将电脑和教学有机地结合在一起。主要功能有人脸识别签到，师生共享屏幕，录屏，课堂反馈，资源共享，课堂小测。我们希望通过该软件增加课上的师生互动，提高学生的听课效率从而提升教学质量。
